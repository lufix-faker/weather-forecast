# weather-forecast

#### 介绍
使用vue框架做的一个移动web版天气预报。

#### 使用说明

1.  cd 到项目根目录
2.  npm install 安装相关依赖包
3.  npm run serve 运行项目
4.  浏览器查看效果（推荐Chrome）
